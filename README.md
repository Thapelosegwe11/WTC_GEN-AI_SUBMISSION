# WTC_GEN-AI_SUBMISSION

## Project Purpose
This project aims to harness the capabilities of Generative AI for various applications. It explores the potentials of AI technologies to create, optimize, and enhance user experiences across different domains.

## Project Structure

### Directory Layout
- **/src** - This directory contains all the source code for the project. It is where the main application logic resides.
- **/tests** - Unit and integration tests can be found here. This ensures that the application functions as expected.
- **/docs** - Documentation for the project, including setup and usage instructions.
- **README.md** - This file provides an overview of the project.

### Key Files
- **app.py** - Main application entry point.
- **requirements.txt** - List of dependencies required for the project.
- **setup.py** - Script for installing the package.

## Getting Started
To get started with the project, follow these steps:
1. Clone the repository:
   ```
   git clone https://github.com/Thapelosegwe11/WTC_GEN-AI_SUBMISSION.git
   ```
2. Navigate to the project directory:
   ```
   cd WTC_GEN-AI_SUBMISSION
   ```
3. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
4. Run the application:
   ```
   python app.py
   ```
